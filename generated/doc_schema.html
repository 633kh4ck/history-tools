Function eosio::schema_type_name

template <typename T>
std::string_view schema_type_name(T*);

Get JSON Schema type name. The first argument is ignored; it may be nullptr.

Returns “”, which prevents the type from being in the definitions section of the schema.

Function eosio::make_json_schema

template <typename T>
void make_json_schema(T*, std::vector<char>& dest);

Convert types to JSON Schema. Appends to dest. The first argument is ignored; it may be nullptr.

This overload works with reflected objects.

Function eosio::make_json_schema

template <typename T>
std::vector<char> make_json_schema(T* = nullptr);

Convert types to JSON Schema and return result. This overload creates a schema including the definitions section. The other overloads assume the definitions already exist.
